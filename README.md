# sigmutselcovs

**Tools for creating covariate matrices for mutation rate modeling**

## Overview

Companion tools for building covariate matrices used in mutation rate modeling with [sigmutsel](https://github.com/alfaromurillo/sigmutsel). `sigmutselcovs` provides code to process and integrate various genomic features including:

- Gene expression data (GTEx, TCGA)
- Chromatin accessibility (ATAC-seq)
- Histone modifications (ChIP-seq)
- Replication timing
- Principal component analysis for dimensionality reduction

## Related Packages

- [sigmutsel](https://github.com/alfaromurillo/sigmutsel): Mutation rate estimation and selection inference
